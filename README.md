# Brain-Tumor-Detection-Project
این پروژه یک وب‌اپلیکیشن هوشمند برای تشخیص و طبقه‌بندی خودکار تومورهای مغزی از روی تصاویر MRI است. سیستم بر پایه **شبکه‌های عصبی کانولوشنی (CNN)** و تکنیک **یادگیری انتقالی  پیاده‌سازی شده است. برخلاف نمونه‌های شبیه‌سازی شده، این پروژه از یک مدل آموزش‌دیده واقعی  استفاده می‌کند که قادر است تصاویر جدید را با دقت بالا پردازش و تحلیل کند.
